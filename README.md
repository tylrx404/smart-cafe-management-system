🌱 EcoPulse – Green Tech Web Platform
EcoPulse is a Green-Tech web application designed to promote environmental awareness and sustainable living.
It provides real-time weather and air quality insights, combined with daily user activity tracking, to help individuals, communities, and authorities understand and reduce their environmental impact.
🚀 Problem Statement
Air pollution, climate change, and unsustainable daily habits are growing concerns in India and across the world.
Most existing platforms only show data, but do not help users connect their daily lifestyle choices with environmental impact.
EcoPulse bridges this gap by:

Showing live environmental data
Collecting daily user inputs
Converting them into meaningful analysis and insights
💡 Solution Overview
EcoPulse acts as a digital pulse of the environment, combining:
🌦 Weather data
🌫 Air Quality Index (AQI)
🧠 User daily behavior
📊 Trends & analytics
🗺 Geographical heatmaps
This helps:
Travelers plan safer trips
Residents improve daily habits
Governments & institutions analyze environmental patterns
✨ Key Features
🔐 Authentication
Simple login with city selection
User-based data tracking
📊 Dashboard
Real-time temperature, humidity, wind speed, UV index
Live AQI with health categories
Green Index Score (eco-friendly rating)
📝 Daily Input (Unique Feature)
Users log daily habits such as:
AC/Fan usage
Water consumption
Outdoor exposure
Transportation mode
Waste segregation
➡ These inputs are locked once submitted and used for analysis.
📈 Analytics
Personalized eco-score calculation
Habit-based impact insights
Progress tracking over time
🗺 India Heatmap
Interactive map showing pollution intensity
Helps users visually understand regional pollution
📉 Trends
Environmental trends based on time and location
Identifies improvement or degradation patterns
🚯 Tackle Pollution
Awareness-based solutions
Practical actions users can take to reduce pollution
⚠ Major Pollutions
Educational section explaining:
PM2.5, PM10, CO, NO₂, SO₂, O₃
Health effects and risks

Frontend: Next.js (App Router), React, TypeScript
Styling: Tailwind CSS
Icons & UI: Lucide Icons, ShadCN UI
State Management: React Hooks
APIs Used:
OpenWeather API (Weather data)
OpenWeather Air Pollution API (AQI & pollutants)
Version Control: Git & GitHub
